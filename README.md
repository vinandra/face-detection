## face-detection-openCV

a simple face detection using python

## Getting Started

1.make virtual environment

```bash
python3 -m venv venv
```

2. Activate virtual env
   (Linux/macOS)

```bash
source venv/bin/activate
```

3.install library

```bash
pip install -r requirements.txt
```
